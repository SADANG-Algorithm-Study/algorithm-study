## 1. [리코쳇 로봇(Lv2)](https://school.programmers.co.kr/learn/courses/30/lessons/169199)
    
**문제 분석**
- 한 번 이동할 때 장애물에 부딪히거나 끝까지 이동
- output :시작 위치에서 목표 위치까지 최소 이동수 or -1(불가능)

**문제 풀이**
1. board 2차원 배열로, start 위치 저장
2. bfs 
    - queue에 x, y, count 저장
    1. 상하좌우로 장애물, board 끝까지 이동
    2. end 위치에 도달하면 종료
